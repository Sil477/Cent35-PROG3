
# Ejemplo de API: PokeAPI

Este proyecto es un ejemplo práctico de cómo interactuar con una API pública utilizando HTML, CSS, JavaScript y PHP. La API utilizada es la **PokeAPI**, que permite acceder a información sobre Pokémon de manera gratuita.

## ¿Qué es PokeAPI?

[PokeAPI](https://pokeapi.co/) es una API pública que proporciona datos sobre diferentes Pokémon, incluyendo sus habilidades, movimientos y características. Puedes acceder a esta información a través de solicitudes HTTP en formato JSON.

## Estructura de la URL

Para obtener información sobre un Pokémon específico, se utiliza la siguiente estructura de URL:

https://pokeapi.co/api/v2/pokemon/{id o nombre del Pokémon}/
Cómo utilizar este proyecto
Clona el repositorio en tu máquina local.
Ejecuta el servidor en tu entorno de desarrollo local (por ejemplo, usando XAMPP).
Accede al archivo index.html en tu navegador para ver cómo se presenta la información de Pokémon de forma amigable.
Modifica el código según tus necesidades para explorar más funcionalidades de la PokeAPI.
